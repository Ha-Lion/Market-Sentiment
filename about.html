About the project

Built to Read the Market Narrative Faster

Public Sentiment Dash is an AI-assisted market sentiment dashboard designed to help users understand what the public financial narrative is saying across major markets.

The website combines public market headlines, source-level sentiment, technical direction, relevant news, and anonymous user sentiment into one cleaner view.

Open Dashboard Send Feedback

Market Experience Behind It

This website is built by people with years of experience following trading, financial markets, economic headlines, and market behavior.

AI-Assisted Intelligence

AI is used to organize public information, improve classification, support cleaner market grouping, and help the dashboard become smarter over time.

Built to Improve

The project is intentionally designed to keep learning. As data quality improves and users give feedback, the dashboard will become more useful and more accurate.

Why Public Sentiment Dash Is Different

Most market news pages show headlines. Public Sentiment Dash is built to go further by organizing those headlines into a practical market-sentiment structure.

Instead of forcing users to scan dozens of sources manually, the dashboard highlights public sentiment, relevant headlines, source votes, user sentiment, and technical direction in one place.

The goal is not to replace judgment. The goal is to help traders, investors, and market watchers see the public narrative faster, then make their own decisions with better context.

User Feedback Matters

This project will keep evolving. If something looks wrong, incomplete, confusing, or worth improving, users are encouraged to send feedback.

Every useful suggestion helps improve the dashboard, the source coverage, the sentiment logic, and the user experience.
