How the Dashboard Works

Public Sentiment Dash turns public financial headlines into a cleaner market-sentiment view. It combines public narrative, relevant news, source votes, technical direction, and anonymous user sentiment.

1

Public Source Collection

The private engine scans public financial, macro, forex, crypto, commodity, stock, and market news sources on a scheduled basis.

2

Rolling News Memory

Relevant headlines are saved into a rolling archive so keyword search can find stories beyond only the latest scan.

3

Market and Instrument Classification

Headlines are grouped by market type and instrument, such as indexes, bonds, forex, crypto, commodities, macro themes, and selected company-news categories.

4

Public Sentiment Index

PSI means Public Sentiment Index. A reading near 0/100 is bearish, around 50/100 is mixed or neutral, and near 100/100 is bullish.

5

Source Votes and Impact

Source votes summarize the filtered public narrative from each source. Impact means a headline contains stronger market-moving keywords; it is not a source-quality score.

6

Technical Direction

When price data is available, the dashboard adds daily, weekly, and monthly technical direction to help compare public narrative against price behavior.

7

Anonymous User Sentiment

Users can vote bullish or bearish on instruments without registration. If voting data is not strong enough yet, the site falls back to daily technical direction so the display stays useful.

Important Note

This dashboard is for information, education, and market awareness. It is not financial advice, not a trading signal, and not a recommendation to buy or sell any asset.
