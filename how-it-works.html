How It Works

How Public Sentiment Dash Works

Public Sentiment Dash scans public financial headlines, classifies market sentiment, organizes instruments, and turns the public narrative into a cleaner dashboard view.

The goal is simple: help readers understand what the public market narrative is leaning toward without digging through dozens of pages manually.

PSI
Public Sentiment Index: 0 bearish, 50 mixed, 100 bullish.
1

Collect Headlines

The system reads public financial sources and collects relevant headlines across stocks, forex, crypto, commodities, bonds, and macro topics.

2

Classify Sentiment

Each headline is classified as bullish, bearish, mixed, or neutral based on its public market meaning.

3

Organize Markets

Headlines are grouped by market, instrument, source, impact, and topic so users can compare the narrative quickly.

4

Display Dashboard

The site shows global PSI, instrument PSI, source votes, user votes, technical direction, and relevant news.

PSI Scale

How to Read the Public Sentiment Index

0–30
Bearish public sentiment
40–60
Mixed or neutral sentiment
70–100
Bullish public sentiment
Data Layers

What the Dashboard Shows

Global PSI: the broad all-market public sentiment score.
Instrument PSI: sentiment for selected assets, pairs, commodities, indexes, crypto, bonds, and macro themes.
Source votes: how individual public sources lean based on matching headlines.
User sentiment: anonymous bullish or bearish reader votes.
Relevant news: public headlines connected to the selected market or instrument.
Technical direction: a simple price-based direction layer when chart data is available.
Important Limits

What It Does Not Do

No financial advice: the dashboard does not tell users what to buy, sell, short, hold, or trade.
No perfect prediction: sentiment can be wrong, crowded, delayed, emotional, or incomplete.
No private account data: the site does not need brokerage login information or personal trading account details.

Use the Dashboard

Open the dashboard to compare instruments or browse the news feed to search public market headlines.
