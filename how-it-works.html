How the dashboard works

Public Sentiment Dash turns large public headline flow into a cleaner market-sentiment view.

1

Public source collection

The engine scans configured public financial, macro, commodity, forex, crypto, and market news sources.

2

Headline filtering

Collected headlines are filtered and tagged by markets, instruments, and relevance so the dashboard stays focused.

3

Sentiment grouping

The engine groups the public narrative behind headlines and labels it as bullish, bearish, mixed, or neutral when enough context is available.

4

Source votes

Each readable source contributes a source-level public sentiment view so users can see which sources are leaning in which direction.

5

Dashboard display

The public website displays the latest summary, source votes, market focus, relevant headlines, and technical direction in one place.
