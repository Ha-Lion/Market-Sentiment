How to Read This Page

PSI

0/100 is very bearish, 50/100 is mixed or neutral, and 100/100 is very bullish public sentiment.

User Sentiment

Anonymous daily user voting. If votes are not strong enough yet, it falls back to daily technical direction.

Impact

Impact means the headline contains stronger market-moving keywords. It is not a source-quality score.

Technical Direction

Technical direction comes from price data when available. Macro themes may show “No price chart.”
